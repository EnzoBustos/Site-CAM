Perguntas Frequentes

  • O que faz o CAM ?

    O CAM é Centro Acadêmico dos alunos de Engenharia Mecânica e Mecatrônica, sendo assim responsável por representar os discentes frente aos professores.

  • O que é um RD ?

    RD é um aluno que foi votado para ser Representante Discente, sendo assim ele carrega a voz dos alunos para dentro de Conselhos e Reuniões com os professores.

  • Como que eu consigo uma IC ?

    Para conseguir uma Iniciação Científica você deve falar com um professor que tem alguma pesquisa em andamento, para isso confira nosso Guia de IC

  • O que é PAPFE?

    PAPFE (Programa de Apoio à Permanência e Formação Estudantil) é o programa que integra as políticas de permanência implantadas pela universidade na USP. Essas politicas são voltadas para os alunos com dificuldades socioeconômicas para se manter na faculdade, oferecendo ajuda financeira com alimentação, moradia, material escolar entre outros benefícios

  • Como faço para pedir um requerimento?

    Para realizar um requerimento basta entrar no jupiter, lá estara presente a opção requerimento. Fique atento as datas pois para realizar requerimento existe um periodo específico.

  • O que são as materias do biênio?

    São chamadas de matérias do biênio as matérias comuns a todas as engenharias, como, por exemplo, Cálculo, Física, Álgebra linear, entre outras. Essas matérias são ministradas ao longo dos dois primeiros anos da Engenharia no prédio chamado Biênio, que é localizado dentro da Poli
